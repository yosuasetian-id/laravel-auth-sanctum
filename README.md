<h1 align="center">Laravel Basic Authentication API</h1>

<p align="center">A clean and simple authentication system built with Laravel and Sanctum.
This project demonstrates basic authentication including register, login, and logout using API tokens.</p>

<hr/>

## Tech Stack
```text
1. Laravel 12
2. Laravel Sanctum
3. MySQL / PostgreSQL (MySQL)
4. REST API
```

## Installation
```text
git clone https://github.com/yosuasetian-id/laravel-auth-sanctum.git

cd laravel-auth-sanctum

composer install
```
## Environment Setup
```text
cp .env.example .env

php artisan key:generate
```
```text
* Set Database

DB_CONNECTION=your_connection
DB_HOST=your_host
DB_PORT=your_port
DB_DATABASE=your_db
DB_USERNAME=your_username
DB_PASSWORD=your_password
```
